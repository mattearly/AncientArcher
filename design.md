# Ancient Archer (TM)

Author(s): Matthew Early;  Reviewer(s): Matthew Early  
Details Last Updated: see timestamp on github

## Overview

Ancient Archer (TM) is an open source game engine to make coding up a game from *scratch* a little easier.

## Context

3D/2D Renderering, Model Loading, Point, Directional, and Spot Lights, Math, Timing, Fully Mutating Objects (scale, translate, rotate), ~Animations~, ~Physics~, Controls, Window Display, Cameras, GLSL Shaders, Lighting, Cubemaped Skyboxes, and 3D Sound.

## C++ Conventions

**naming conventions**
Public functions AreCapsLikeThis
Private functions camelCase
Experimental, depricated or temporary functions being with __

Private member variables begin with m
Public member variables should not exist, instead use getters and setters with apporpriate restrictions.
